The objective is to produce proofs in lean for unsatisfiable problems. This means that the solver will have a flag –proof file  that will  write a proof into the file if the given formula was successfully shown to be unsatisfiable.

The proof will be a single  theorem with possibly lemmas inside.  Since we are dealing with QF_UF  we will simply rely on lean’s built-in  setup, i.e. equality and function  applications.
 So the first task  to produce the proof will require translating the input SMT  problem into LEAN.
